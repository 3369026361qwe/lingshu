## 五、统计显著性检验

### IC t 检验

对每个活跃因子的 IC 序列做单样本 t 检验：

```
H0: IC 均值 = 0（因子无预测能力）
H1: IC 均值 ≠ 0
```

当前已有个别因子 IC 的均值数据，需要添加：
- t 统计量 = IC_mean / (IC_std / sqrt(n))
- p 值
- 汇报形式：表列出因子名、IC 均值、t 值、p 值

### GNN 精度显著性

```
H0: GNN Top-5 精度 = 1/5 = 20%（随机猜测——5000 只里瞎选 5 只）
H1: GNN Top-5 精度 > 20%
```

当前 GNN Top-5 精度 87.8%远超随机水平 20%，但需要形式化：
- 二项检验：87.8% vs 20%，n = 测试快照数
- 汇报形式：p < 0.001
